Tentang Saya

Mengubah Data Menjadi Dampak Nyata

Saya M. Gymnastiar — seorang ML engineer dan pendidik asal Banjarmasin, Kalimantan Selatan. Saya merupakan lulusan S1 Pendidikan Komputer dari Universitas Lambung Mangkurat (IPK: 3.72/4.00), dan saat ini sedang memperdalam keahlian melalui program Magister Ilmu Komputer di Universitas Gadjah Mada.

Pekerjaan saya berada di persimpangan antara AI terapan dan pendidikan — saya membangun sistem ML siap produksi sekaligus membimbing generasi praktisi berikutnya. Saya telah membimbing 20+ peserta secara remote melalui DBS Foundation Coding Camp, membangun sistem rekomendasi, pipeline analitik prediktif, dan platform pembelajaran interaktif.

Saya adalah dua kali finalis nasional GEMASTIK (kompetisi ICT terbesar untuk mahasiswa Indonesia), penerima insentif PKM, dan peraih penghargaan Peserta Terbaik di PT GITS Indonesia. Saya aktif mencari keterlibatan remote fleksibel di samping studi pascasarjana saya.
